NAME
       castool - MESS Generic cassette manipulation tool

SYNOPSIS
       castool convert format infile outfile.wav

DESCRIPTION
       castool	is  a  tool  for  the maintenance and manipulation of cassette
       dumps that MESS users need to deal with.

       convert
